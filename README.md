## Featured Projects

### ShadowBlade Dice Shop

A full-stack e-commerce demo for a handmade dice shop concept. ShadowBlade Dice Shop features a React and Vite frontend, an Express backend API, product inventory, a shopping cart, demo checkout flow, order confirmation screen, image gallery/detail view, and responsive dark fantasy-inspired styling.

**Tech:** React, Vite, JavaScript, CSS, Node.js, Express, GitHub Pages
**Live Demo:** https://morganneedham.github.io/shop/
**GitHub:** https://github.com/MorganNeedham/shop

### Roll Oracle

A React and TypeScript dice analytics app for tabletop players. Roll Oracle tracks dice profiles, saves roll history, calculates roll statistics, and rates whether dice are blessed, cursed, or suspiciously normal.

**Tech:** React, TypeScript, Vite, CSS, GitHub Pages
**Live Demo:** https://morganneedham.github.io/roll-oracle/
**GitHub:** https://github.com/MorganNeedham/roll-oracle

### Portfolio Website

A responsive personal portfolio website built to showcase front-end development projects, technical skills, resume information, and contact links. The site highlights deployed projects and was designed with a clean, accessible, job-application-ready layout.

**Tech:** HTML, CSS, JavaScript, GitHub Pages
